# Tutorial - User Guide

This tutorial will guide you through all the features of FastUI, from basic to advanced.

The tutorial is divided into several sections:

- [First Steps](first-steps.md) — Installation and basic concepts
- [Components](components.md) — All built-in UI components
- [Routing](routing.md) — URL patterns with typed parameters

### Core Features

- [Server Actions](server-actions.md) — POST callback handlers
- [Layout and Navigation](layout-and-navigation.md) — Multi-page apps with shared navigation
- [Customization](customization.md) — CSS, stylesheets, inline styles

### Advanced

- [OpenAPI](openapi.md) — Swagger UI configuration and tags
- [Hot Reload](hot-reload.md) — Auto-refresh on file changes
- [Examples](examples.md) — Complete application examples

## How to Use

Each section builds upon the previous one. We recommend reading in order if you are new to FastUI.

All sections include practical code examples that you can copy and run. Full examples are also available in the [examples directory](https://github.com/ndugram/fastui2/tree/master/examples).
