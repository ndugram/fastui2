# Learn

This section covers fundamental concepts and background information to help you understand FastUI better.

## Available Topics

- [Python Type Annotations](python-types.md) — understanding type hints used in FastUI

## Prerequisites

FastUI is designed to be easy to use, but basic knowledge of Python is required:

- Python 3.10+
- Basic understanding of HTTP and web servers
- Familiarity with decorators in Python

## What You Will Learn

After going through the documentation, you will know:

- How to define pages using decorators
- How to use all built-in UI components
- How to create URL patterns with typed parameters
- How to implement server-side actions (POST handlers)
- How to customize styles with CSS
- How to configure OpenAPI documentation
- How to use hot reload for development
