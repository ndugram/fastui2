# Справочник

Полный API reference для FastUI.

## Классы

- [App](app.md) — главный класс приложения
- [Component Protocol](../tutorial/components.md#протокол-component) — интерфейс компонента

- [Components](components.md) — все встроенные UI-компоненты

## Быстрые ссылки

| API | Описание |
|---|---|---|
| `App()` | Создать приложение FastUI |
| `@app.page()` | Зарегистрировать маршрут |
| `@app.setter()` | Назначить маршрут целью редиректа с 404 |
| `app.action()` | Зарегистрировать обработчик действия |
| `app.run()` | Запустить dev-сервер |
| `ui.heading()` | Создать заголовок |
| `ui.text()` | Создать текст |
| `ui.button()` | Создать кнопку |
| `ui.input()` | Создать поле ввода |
| `ui.link()` | Создать ссылку |
| `ui.code()` | Создать блок кода |
| `ui.divider()` | Создать разделитель |
| `ui.page()` | Создать контейнер |
