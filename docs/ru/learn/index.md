# Изучение

В этом разделе собраны фундаментальные концепции для понимания FastUI.

## Темы

- [Аннотации типов Python](python-types.md) — понимание type hints, используемых в FastUI

## Предварительные требования

- Python 3.10+
- Базовое понимание HTTP и веб-серверов
- Знакомство с декораторами в Python

## Чему вы научитесь

- Определять страницы через декораторы
- Использовать все встроенные UI-компоненты
- Создавать URL-паттерны с типизированными параметрами
- Реализовывать серверные действия (POST-обработчики)
- Настраивать стили через CSS
- Конфигурировать OpenAPI-документацию
- Использовать hot reload для разработки
