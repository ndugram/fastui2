# Примеры

Полные работающие примеры доступны в [директории examples](https://github.com/ndugram/fastui2/tree/master/examples).

## Список примеров

| # | Файл | Что демонстрирует |
|---|---|---|
| 1 | hello_world.py | Минимальное приложение |
| 2 | all_components.py | Все встроенные компоненты |
| 3 | route_params.py | URL-паттерны с параметрами |
| 4 | server_actions.py | POST-обработчики |
| 5 | custom_css.py | Тёмная тема |
| 6 | forms.py | Поля ввода |
| 7 | navigation.py | Многостраничная навигация |
| 8 | docs_config.py | Настройка OpenAPI |
| 9 | layout_page.py | Page для группировки |
| 10 | counter.py | Интерактивный счётчик |
| 11 | todo.py | Todo-приложение |
| 12 | hot_reload.py | Hot reload |
| 13 | multi_page.py | Общая навигация |
| 14 | no_docs.py | Без документации |
| 15 | external_stylesheets.py | Bootstrap |
| 16 | single_component.py | Один компонент |
| 17 | advanced_routing.py | Сложные маршруты |
| 18 | inline_styles.py | Комбинации стилей |
| 19 | minimal.py | Минимальное приложение (7 строк) |
| 20 | dynamic_routes.py | Динамические маршруты |

## Запуск

```bash
git clone https://github.com/ndugram/fastui2.git
cd fastui
python examples/01_hello_world.py
```
